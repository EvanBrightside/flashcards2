# FLASHCARDS 2

## Setup
* git clone `git@github.com:EvanBrightside/flashcards2.git`
* bundle install
* config database.yml
* rails db:setup
* rails db:migrate

## Tech
* Ruby on Rails 5.1.6
* Ruby 2.4.3

## Usage
* rails s
* localhost:3000
